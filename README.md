# VedNetra

Static GitHub Pages deployment package for VedNetra Advanced v84.

## Deploy to GitHub Pages

1. Create a new public GitHub repository, for example `vednetra`.
2. Upload these files to the repository root, or push this folder with Git.
3. In GitHub, open the repository Settings.
4. Go to Pages.
5. Set Source to `Deploy from a branch`.
6. Select branch `main` and folder `/root`.
7. Save.

The site will be available at:

`https://YOUR-GITHUB-USERNAME.github.io/vednetra/`
