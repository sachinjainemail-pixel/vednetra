First-time setup (one minute):
  1. On the TV: Settings → General → Network — note the TV's IP address and make sure it's on the same Wi-Fi as your phone.
  2. Enable Mobile TV On / LG Connect Apps in the TV's network settings (lets phones control it).
  3. If this page is served over https, tap ① Accept TV certificate, then on the warning page choose Advanced → Proceed. (The TV uses a self-signed cert — this is safe on your own network.)
  4. Type the IP above and tap Connect. A pairing prompt appears on the TV — select Accept with the physical remote once. After that it reconnects automatically.
